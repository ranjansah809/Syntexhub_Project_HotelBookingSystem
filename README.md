# Hotel Booking Management System

A Java CLI-based Hotel Booking Management System developed during the Syntexhub Internship Program.

## Features
- User Login Authentication
- View Available Rooms
- Book Room
- Cancel Booking
- Menu Driven Interface

## Technologies Used
- Java
- OOP Concepts
- Scanner Class
- Arrays / Classes

## Project Structure
- Main.java
- HotelManagement.java
- Room.java
